What It Does
Everything PF-LU Needs in One Place

From field recording to admin reporting — all roles, all features, one system.

Trap Network Management

Admins create and manage trap lines and individual traps with GPS coordinates, aligned to the trap.nz format.

Field Data Entry

Operators log trap check results including species caught, trap status, bait type, and condition directly from the field.

Reporting & Analytics

Visual charts and filterable tables give everyone a clear view of predator activity trends across all trap lines.

CSV Export

Download all catch record data as a CSV file — formatted to be compatible with trap.nz's import requirements.

Role-Based Access

Three roles — Observer, Operator, and Admin — each with the right level of access for their responsibilities.

Mobile Friendly

Fully responsive design so field operators can log data directly from their phones while checking traps on campus.

How It Works
Simple for Everyone in the Field
1
Register & Log In

Create an account as an Observer. An Admin upgrades your role to Operator if you're doing fieldwork.

2
Check Your Assigned Line

Operators are assigned to specific trap lines by the Admin. Head out, check each trap along your route.

3
Log the Results

Record what you found — species caught, trap status, bait used, condition. Data goes straight to the database.

Access Levels
Three Roles, One System
{# Observer #}
👁 Observer

Volunteer, trainee, or visitor

  • ✅ View trap lines and traps
  • ✅ Browse catch records
  • ✅ Download CSV data
  • ✅ View charts and reports
{# Operator — highlighted #}
🔧 Operator Field Worker

Field worker, ranger, or employee

  • ✅ Everything Observers can do
  • ✅ Add new catch records
  • ✅ Edit own catch records
  • ✅ Record incidental observations
  • ✅ View assigned lines
{# Admin #}
⚙️ Admin

System administrator

  • ✅ Everything Operators can do
  • ✅ Create & manage trap lines/traps
  • ✅ Assign Operators to lines
  • ✅ Manage all users & roles
  • ✅ Manage lookup data
